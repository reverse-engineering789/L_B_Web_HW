GitHub pages

Websites for you and your projects.

Host Irectly from your GitHub repository. Just edit, push, and your changes are live

Ready to get started? Build your own site
from scratch or generate one for your
project

You get one site per GitHub account and orgainization,
and unlimited project sites. Let's get started.

Create a repository

over to GitHub and create a new public repository
named username..github.io, where username is your
username (or orgainization name) or GitHub

if the first part of the repository doesn't exactly match your username, it won't
work, so mae sure to get it right

What git client are you using?

Clone the repository

Got the folder where you want to store your project and clone the new repository

Hello world

Enter the project folder and add an index.html file.

Push it

Add, commit and push your changes

...and you are done!

Fire up a browser and go to https://username.github.io
